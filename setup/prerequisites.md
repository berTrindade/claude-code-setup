# Prerequisites

Required tools and versions for Claude Code setup.

## Required Tools

### Claude Code
- **Claude Code CLI** - Latest version
- Install: Follow official installation guide

### Development Tools
- **Node.js** - 18+ (for npm/npx commands)
- **Git** - Latest version
- **Docker** - Latest version (for Terraform MCP server)
- **uv** - Python package manager (for aws-docs MCP server)

## Environment Variables

Set these in your shell or .env file (not committed):
- GITHUB_PERSONAL_ACCESS_TOKEN
- POSTHOG_PERSONAL_API_KEY (if using)
- DATABASE_URL (if using postgres MCP)
